# boringhub

这是一个基于 Next.js 15 构建的简单网站项目。

## 项目简介

该网站展示了一个极简风格的页面，主要内容为"我很无聊 / I AM SO BORED"，采用电影标题式的视觉设计。网站目前处于建设中状态，具有简单的响应式布局和现代化的 UI 设计。

## 技术栈

- [Next.js 15](https://nextjs.org/)
- React 19
- TypeScript 5
- Tailwind CSS 4
- Wrangler (Cloudflare Workers)

## 开发环境搭建

### 前置要求

确保您的开发环境中已安装以下软件：

- Node.js (推荐版本 18 或更高)
- pnpm 包管理器

### 安装步骤

1. 克隆项目到本地：
   ```bash
   git clone <repository-url>
   cd boringhub
   ```
